## 开发

参照`.env.example`，新建`.env.development`文件，配置好环境变量。

```bash
yarn start
```

## 部署

参照`.env.example`，新建`.env`文件，配置好环境变量。

```bash
yarn build
```

`build`文件夹中即为编译后的静态资源文件。
